# Triage Labels

Agent skills 使用五个标准 triage role。本文档约定这些 role 在本仓库 issue tracker 中对应的 label 字符串。

| mattpocock/skills 中的 label | 本仓库 tracker 中的 label | 含义 |
| ---------------------------- | ------------------------- | ---- |
| `needs-triage`               | `needs-triage`            | 需要维护者评估 |
| `needs-info`                 | `needs-info`              | 等待补充信息 |
| `ready-for-agent`            | `ready-for-agent`         | 规格已经足够，agent 可以接手 |
| `ready-for-human`            | `ready-for-human`         | 需要人类实现或处理 |
| `wontfix`                    | `wontfix`                 | 不会处理 |

当 skill 提到某个 role 时，使用上表中对应的 label 字符串。
